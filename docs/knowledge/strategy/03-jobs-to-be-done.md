# Jobs To Be Done (JTBD)

**The Platform (EKB)**

## Primary JTBD (Core Problem Solved)

"When I run operations involving physical spaces, events, or logistics, I want to execute bookings, issue access, process payments, and track everything so that I can trust the records, avoid conflicts, comply with audits, and scale without chaos."

**Functional:**

- Manage space availability without double-booking.
- Issue verifiable access rights.
- Record financial transactions immutably.
- Orchestrate workflows with approvals.
- Query knowledge and get accurate AI assistance.

## Secondary JTBD

- For Finance: Reconcile and report without errors.
- For Compliance: Prove sovereignty and audit trail.
- For Operations: Real-time visibility into utilization.
- For IT/Dev: Integrate without lock-in, extend safely.

## Emotional JTBD

- Peace of mind: "I know the truth is in the ledger."
- Control: "I own my data and rules."
- Confidence: "AI helps but doesn't decide money or access."
- Professionalism: "The system feels serious and efficient, not flashy."

## Social JTBD

- For the organization: Demonstrate operational maturity to partners, auditors, customers.
- For teams: Reduce friction in handoffs between spatial, commerce, finance.
