# Licensing Analysis

**Platform**: AI-Native Project & Event Operations Platform

## Recommended

- Core OSS: AGPL-3.0 with additional terms (like Hi.Events/Pretix) to allow self-host but require source for hosted mods.
- Commercial: For removing copyleft or enterprise features.
- Client libs: MIT/Apache.

## Risks/Benefits

- Benefits: Attracts OSS users, community (like ERPNext, Plane).
- Risks: Copyleft may deter some SaaS; mitigate with clear commercial path.

Inspired by successful OSS event/PM tools.
