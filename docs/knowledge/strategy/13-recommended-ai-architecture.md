# Recommended AI Architecture

**Platform**: AI-Native Project & Event Operations Platform

## Core

- RAG over knowledge base + structured ops data.
- Agents: Scheduling assistant, resource optimizer, document extractor, budget forecaster, post-event analyzer.
- Guardrails: Human approval for material actions (L-06 style).

## UI Integration

- Contextual in ops views (not separate chat).

Inspired by Glean (enterprise AI), Cursor/Linear AI (in-product), agent frameworks.

See research for details.

## Integration with Ops

AI helps with the "missing opportunities" like conflict resolution, optimization.
