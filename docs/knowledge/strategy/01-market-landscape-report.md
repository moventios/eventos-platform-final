# Market Landscape Report

**Focus**: AI-Native Project & Event Operations Platform

## Summary from Research

The market for event and project management is fragmented:
- Event-specific (Eventbrite, Pretix, Hi.Events, Cvent): Strong on ticketing/registration, weak on integrated project execution, finance, AI.
- Project/PM tools (Plane, OpenProject, Monday.com, Asana, Linear): Excellent for tasks/boards, missing venue/resource booking, event commerce, immutable audit.
- ERP (Odoo, ERPNext): Broad but bloated, not focused on events/projects or AI-native.
- AI tools (Glean, AppFlowy): Great for search/knowledge, not operations backbone.

**Opportunity for this Platform**:
- Unified: Event + Project + Resource + Finance + AI in one auditable system.
- AI-Native: Agents for scheduling, optimization, documentation.
- Open + Self-host friendly + hosted Moventios.
- Focused on "operations" not just planning: execution, approvals, audit, intelligence.

ICP: Event organizers, PMOs, venue managers, corporate event teams, production companies. Mid-market to enterprise, hybrid physical/digital ops.

## Key Insights
- Pricing: Freemium/OSS core + per-event/seat/usage + enterprise.
- GTM: Self-serve + partners + OSS community.
- Differentiation: AI deeply in workflows, audit-grade, open architecture.
