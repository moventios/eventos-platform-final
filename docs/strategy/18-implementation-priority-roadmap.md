# Implementation Priority Roadmap

**Platform**: AI-Native Project & Event Operations Platform

## Phase 1: MVP (Core Ops)
- Event/project CRUD, scheduling, basic resources, simple finance.
- Basic AI RAG.
- Stoic UX for key views.
- Self-host basics.

## Phase 2: Intelligence (6-12 mo)
- AI agents for suggestions, conflict resolution, optimization.
- Advanced workflows/approvals.
- Better audit views.
- SaaS hosting.

## Phase 3: Ecosystem
- Marketplace.
- Vertical packs.
- Advanced AI (forecasting, analysis).
- Enterprise features (white-label, SLAs).

Prioritized by customer impact on ops pain points, build vs buy, from benchmarks.

See research for rationale.
